# PitchNames

## Introduction

**PitchNames** is a simple Instrument showing [PitchNames](https://steinbergmedia.github.io/vst3_dev_portal/pages/Technical+Documentation/Presets+Program+Lists/Index.html?highlight=PitchNames#pitch-names) support.

> See also: [Online Documentation](https://steinbergmedia.github.io/vst3_dev_portal/pages/What+is+the+VST+3+SDK/Plug-in+Examples.html#pitchnames).

## Getting Started

This plug-in is part of the VST 3 SDK package. It is created with the VST 3 SDK root project.

> See the top-level README of the VST 3 SDK: https://github.com/steinbergmedia/vst3sdk.git
