# mda plug-ins

## Introduction

**mda plug-ins** are FX plug-ins based on the OpenSource mda plug-ins (http://mda.smartelectronix.com/), this set of plug-ins demonstrates how wrap DS- code in a VST 3 plug-in.

> See also: [Online Documentation](https://steinbergmedia.github.io/vst3_dev_portal/pages/What+is+the+VST+3+SDK/Plug-in+Examples.html#mda-plug-ins).

## Getting Started

This plug-in is part of the VST 3 SDK package. It is created with the VST 3 SDK root project.

> See the top-level README of the VST 3 SDK: https://github.com/steinbergmedia/vst3sdk.git
