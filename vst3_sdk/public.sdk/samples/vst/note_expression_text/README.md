# note-expression-text

## Introduction

**note-expression-text** is a simple Instrument plug-in visualizing the [note expression](https://steinbergmedia.github.io/vst3_dev_portal/pages/Technical+Documentation/Change+History/3.5.0/INoteExpressionController.html) as text.

> See also: [Online Documentation](https://steinbergmedia.github.io/vst3_dev_portal/pages/What+is+the+VST+3+SDK/Plug-in+Examples.html#note-expression-text).

## Getting Started

This plug-in is part of the VST 3 SDK package. It is created with the VST 3 SDK root project.

> See the top-level README of the VST 3 SDK: https://github.com/steinbergmedia/vst3sdk.git
