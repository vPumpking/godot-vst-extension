# Test Multiple Program Changes

## Introduction

**Test Multiple Program Changes** is simple FX plug-in showing how to support Program List.

> See also: [Online Documentation](https://steinbergmedia.github.io/vst3_dev_portal/pages/What+is+the+VST+3+SDK/Plug-in+Examples.html#test-program-change).

## Getting Started

This plug-in is part of the VST 3 SDK package. It is created with the VST 3 SDK root project.

> See the top-level README of the VST 3 SDK: https://github.com/steinbergmedia/vst3sdk.git
